<h2>How to run:</h2>

<h3>Clone the repository:</h3>

```
$ git clone https://github.com/unverzed/linkr-front:git
```

<h3>Install dependencies:</h3>

```
$ npm or yarn install
```

<h3>Initializing the API:</h3>

```
$ npm start
```
